# تعديلات واتساب وتصحيح موقع العميل

- إضافة زر واتساب إلى شاشة تفاصيل الشحنة.
- إضافة زري الاتصال وواتساب إلى شاشة تحديث الحالة.
- إنشاء رسائل عربية متغيرة حسب اسم العميل والمتجر ورقم الشحنة.
- إظهار مبلغ التحصيل فقط عندما يكون أكبر من صفر.
- إرسال الرسالة الجاهزة مرة واحدة لكل رقم شحنة، ثم فتح المحادثة بدون رسالة.
- إضافة مهلة 10 ثوانٍ بين عمليات فتح واتساب الناجحة.
- إضافة تصحيح موقع العميل من إحداثيات أو رابط Google Maps، بما فيها الروابط المختصرة.
- حفظ الموقع المصحح محليًا حسب رقم الشحنة.
- إضافة الرجوع للموقع الأصلي من نافذة التصحيح.
- عرض الموقع المصحح على خريطة الشحنات بلون بنفسجي.

ملاحظة: لم يتم تعديل أي API خاص بحالات SLS أو OTP أو التسليم.
