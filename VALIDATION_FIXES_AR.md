# تقرير التحقق

تم تنفيذ الفحوصات التالية داخل بيئة العمل:

- تحليل صياغة جميع ملفات Dart في `lib/` و`test/` باستخدام محلل Tree-sitter: لا توجد أخطاء Syntax.
- التحقق من صحة `AndroidManifest.xml`.
- التحقق من صحة `ios/Runner/Info.plist`.
- التحقق من قراءة `pubspec.yaml`.

لم يتوفر Flutter SDK داخل بيئة العمل، ولذلك يجب تنفيذ الأوامر التالية في Android Studio قبل التثبيت النهائي:

```bash
flutter pub get
flutter analyze
flutter test
flutter run
```

## اختبار ميداني مطلوب

1. سجل الدخول مرة واحدة، أغلق التطبيق بالكامل، ثم افتحه وتأكد أنه يدخل مباشرة.
2. امسح مجموعة شحنات وتأكد أن زر OFD يبقى معطلًا حتى اكتمال العدد.
3. بعد OFD تأكد من الانتقال إلى المهام وظهور الشحنات.
4. اضغط **ابدأ الملاحة** وتأكد من فتح Google Maps مباشرة.
5. تأكد من ظهور اسم المتجر. إذا بقي فارغًا في شحنة معينة، افتح شاشة Debug وأرسل JSON لتحديد اسم الحقل الفعلي الذي يرجعه الحساب.
